Pathfinding Artifical Intelligence - C# Visual Studio

Role - Solo Developer
Description - The aim of this project was to develop a path finding artifical intelligence that finds the shortest route to the player, in this project I developed two different types of algorithms to make this possible, these are A* and LRTA*. The A* algorithm is a graph traversal and path search algorithm, it needs to take in at least 3 parameters to work, these being the player, the bot and the map itself, with these it can calculate the optimal route to the player from the bot keeping in mind any walls that can stop it. The LTRA* alogrithm is similar to the A* alogirthm, the main difference is that it allows the player to see the bot searching for the optimal route in real time. Overall, this project was very interesting and made me passionate about the possibilities with artificial intelligence.
