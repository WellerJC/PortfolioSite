What I've done so far?
Love Letter Card Game - C#, Visual Studio 2019

Role - Solo Developer
This project showcases my proficiency in object-oriented design, incorporating advanced programming principles such as Run-time Polymorphism and Inheritance for a robust and scalable foundation. The implementation features a seamless multiplayer experience, allowing players to engage with up to three friends or AI-driven opponents. The emphasis on a meticulously crafted object-oriented design ensures not only a sophisticated gaming experience but also facilitates future expansion and maintenance.
A key highlight of this project is the integration of an intelligent autosave function. This functionality, triggered at the conclusion of each round, ensures the preservation of game progress. This design choice offers users the convenience of seamlessly resuming their games at their leisure, reinforcing a user-centric approach to application development.
Presented through a console window, the user interface is intentionally text-centric, emphasizing a clean and professional aesthetic. This deliberate design decision enhances accessibility, allowing players to focus on the strategic intricacies of the game.

3D OpenGL Graphics Scene - C#/GLSL Visual Studio

Role - Solo Developer
Description - This project showcases my expertise in the creation and rendering of a 3D scene, achieved without reliance on an existing game engine. Throughout the development of this scene, I meticulously coded essential elements, showcasing a comprehensive skill set that includes draw calls, vertex buffer and array objects, shaders, lighting, input controls, and animation. The scene is set in a room hosting 3 pedestals with different objects on them including an armadillo, each object had different properties, making them unique. For example the armadillo could have its size scaled by the player via keybinds and another had a moon revolving around it. This project not only demonstrated my proficiency in C# but also provided a hands-on experience with GLSL, a coding language integral to shader development. The incorporation of GLSL allowed me to enhance the visual aesthetics of the scene, adding depth and realism through finely tuned shaders.
